# Keybindings (for now)

- in future i will make it have catagories, (like emacs, e.g. C-x,C-c)
- but for now:

| Key | Function |
|-|-|
| `C-h` | Move left |
| `C-j` | Move down |
| `C-k` | Move up|
| `C-l` | Move right |
| `C-v` | Paste to position |
| `C-@` | Toggle selection |
| `C-s` | Save file |
| `C-z` | Next Buffer |
| `C-x` | Prev Buffer |
| `C-w` | Skip word |
| `C-b` | goto previous word |
| `C-e` | end of line |
| `C-^` | start of line |
| `C-q` | exit |
| `\b` | backspace |
| `\n` | new line |
| `*`  | character |
