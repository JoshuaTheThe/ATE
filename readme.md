# Keybindings (for now)
- long keybindings build until non ctrl char
- C-xx is ctrl+xx
- prefixes are: X,C

| Key | Function |
|-|-|
| `C-[` | Move left |
| `C-k` | Move down |
| `C-i` | Move up|
| `C-]` | Move right |
| `C-w` | Next Word |
| `C-b` | Previous Word |
| `C-e` | End of Line |
| `C-^` | Start of line |
| `C-y` | Yank selection |
| `C-p` | Paste |
| `C-v` | Start/End selection |
| `C-x-q` | quit |
| `C-x-c` | close buffer |
| `C-x-n` | next buffer |
| `C-x-p` | previous buffer |
| `C-x-w` | write buffer |
